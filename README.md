# VLSI Projects

A collection of **VLSI Design projects** implemented using **Verilog HDL and SystemVerilog**.
This repository contains various RTL design experiments, digital circuits, simulation projects, and mini-projects developed during VLSI learning and practice.
The projects cover fundamental digital design concepts including combinational logic, sequential circuits, counters, multiplexers, arithmetic units, processors, and communication protocols.

# 07 - VLSI_7

# 4-bit Parallel-In Serial-Out (PISO) Shift Register

The 4-bit Parallel-In Serial-Out (PISO) shift register is a digital circuit that loads 4-bit parallel data when a load signal is asserted and then shifts the bits out serially on each clock pulse.

## Features

- 4-bit parallel data loading
- Serial data output
- Clock-controlled shifting
- Load control operation

## Concepts Covered

- Shift registers
- Sequential logic design
- Data conversion techniques
- Clock-based digital systems
- RTL implementation
